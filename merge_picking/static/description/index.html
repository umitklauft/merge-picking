Delivery Order and Receipt Merger in Odoo

When there are large number of picking orders whether its Delivery Order and Incoming Shipment/Receipt its hard to manage them. Warehouse users need to open each delivery and shipment then process them one by one. Here we provide option to reduce your work, after installing our Odoo apps you can easily merge delivery orders and incoming shipment/Receipt in one click for same customers/suppliers. Just need to select them from tree view (you can use group by partner for easily filter) and merge them. Most important things you can still see different lines for different order on merged pickings and then if you want, you can partially delivered or received it. Once picking has been merge it will create new picking and rest of other pickings get cancelled and source documents of new picking will have reference of cancelled picking, so user can easily get idea from where this new picking get created.

Features

Merge any type of picking.

User can merge any type of picking i.e Delivey Orders, Incoming Shipment, Internal Transfer.

We can not merge one single picking, You should select more than two picking for merge picking.It will raise a warning.


Same Partner Picking Warning


We can not merge different partners picking. It will raise a warning.


Same State Merge Picking Warning

We can merge only same state picking, if we merge different type of state then it will raise a warning.
